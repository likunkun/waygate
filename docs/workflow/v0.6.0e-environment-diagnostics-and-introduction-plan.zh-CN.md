 # V0.6.0e 环境诊断与介绍材料执行单

  ## Summary
  - 版本目标：形成 `0.6.0e`，在 `0.6.0d` 基础上扩展
  `waygate doctor` 环境依赖检测，并补齐推荐环境、配
  置说明、介绍材料和 PPT 大纲。
  - 交付形式：CLI 诊断增强 + Markdown 文档，不生成
  `.pptx`。
  - 范围边界：不进入 V0.6.1 external spec intake，
  不实现 V0.6.2 strict test presence，不提交 `.rrc-
  controller-*` state-dir。

再补一个，需求与验收确认文档中，“需要人确认的点中加入原型页面的确认内容”

  ## Assumptions
  - 介绍材料以中文为主，同时补英文版以匹配当前项目
  双语文档风格。
  - Skills 是否安装到具体 agent runtime 中无法完全
  自动检测，doctor 只提供推荐清单和人工检查提示。
  - `doctor` 是诊断命令，不读取 controller state，
  也不改变任何文件。
